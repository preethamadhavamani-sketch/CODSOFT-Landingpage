🎓 KIT Coimbatore – Responsive College Landing Page

📖 Overview

A modern, responsive, and visually appealing landing page for Kalaignarkarunanidhi Institute of Technology (KIT), Coimbatore. This website is designed to showcase the institution's excellence, academic programs, campus facilities, placements, and contact information with a clean and professional user interface.

✨ Features

- 🏠 Responsive Home Page
- 🧭 Fixed Navigation Bar with Smooth Scrolling
- 🎯 Attractive Hero Section
- 🏛️ About College Section
- 📚 Interactive UG & PG Course Tabs
- 💳 Modern Course Cards
- 📞 Contact & Enquiry Form
- 📱 Mobile-Friendly Navigation Menu
- 🎨 Professional UI Design
- ⚡ Fast Loading & Responsive Layout

💻 Technologies Used

- 🌐 HTML5
- 🎨 CSS3

- 📱 Responsive Web Design

The main objective of this project is to create a professional and responsive college landing page that effectively presents the institution's achievements, academic programs, infrastructure, and admission details while providing visitors with an engaging user experience.
